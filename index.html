删除win7右键菜单intel的图形属性和图形选项

发表于 2019-09-19 分类于 windows


说明 ：如上图，电脑有intel集显的，装好驱动，一般会在桌面右键菜单添加图形属性和图形选项两项(如上图红色框内)，为什么要删除这两项，因为有这两个选项时，每点一次右键，资源管理器进程(explorer.exe)占用的内存都会增加6-8M的内存，最开始explorer.exe也就占用10M多点内存，我们在使用过程中，都会用到右键，或者还会不自觉的右键刷新，这样explorer.exe占用的内存就会越来多，这似乎是intel的一个小bug吧
把右键菜单这两项(图形属性和图形选项)去掉，就没有了这种现象

zabbix在window下安装

发表于 2019-09-18 分类于 zabbix

1.下载好相应的win下的zabbix客户端包
2.解压到某个目录(目录任意，我一般都放在c盘根目录)
3.cmd下进入到zabbix的bin下的win64目录(我的目录是：C:\zabbix_agents_3.0.4.win\bin\win64,如果你安装32位的就进入到win32目录)
4.执行命令：zabbix_agentd -h

5.首先我们要添加到服务，然后再启动(添加到服务方便以后开户自动启动)
在这一步，我们要注意应该如何添加服务，不能直接执行命令：zabbix_agentd.exe -i，虽然这一步不会出问题，但到下一步启动的时候会报错误，还是乱码，情况如下图

经过修改cmd的默认编码得知,错误提示为：服务没有及时响应或控制请求，如下图：

其实就是没找到配置文件而已，所以我们添加服务的时候，应该指定配置文件的路径，用-c参数
zabb-x_agentd.exe -i -c C:\zabbix_agents_3.0.4.win\conf\zabbix_agentd.win.conf
这样再启动，就成功了

再看下任务管理器里，已经有zabbix的进程了

服务也已经启动了

最后说明下我是如何修改cmd的编码的
1.在cmd下执行：chcp 65001
2.打开cmd的属性，切换成字体一项，把字体改为console或者lucida console也可以
3.再执行命令就可以看到正常的中文提示了

要想恢复：
1.先把字体改回来
2.执行chcp 936,完成了

excel打开第二个表格很慢

发表于 2019-09-18 分类于 windows

系统信息：

系统版本：win10 企业版 1607
office版本：office 专业增强版 2016

问题：

excel打开第二个表格很慢，等很久也打不开(2013版本的office也是如此)

原因：

当前文件处于编辑状态下（即光标在单元格内）是不能打开新档的，打开新文档，直接转到当前打开的文档，如果不动，就会一直处于这个状态(我们的感觉就是卡住了)，这时鼠标随便在其它单元格点下，新文件就立马打开了(因为在点的过程中，已经退出了单元格的编辑状态)。所以当我们想打开第二个文件时，要做的就是使单元格不处于编辑状态
